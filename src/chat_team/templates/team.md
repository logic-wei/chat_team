# 团队画像 (Team Profile)

本文件会被原样注入到**每个虚拟员工**每一轮的 system prompt(以 `[团队信息]` 块的形式),
让所有角色都知道自己服务于哪个公司/团队。删空全部内容则不注入(向后兼容)。

修改本文件需重启 `chat-team` 才能生效(配置在启动时一次性读入)。
建议正文不超过 300 字,过长会增加每轮的 API token 消耗。

如果不需要这个能力,把整个文件清空即可。
